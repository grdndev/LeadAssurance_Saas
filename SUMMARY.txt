│  🚀 DÉMARRAGE RAPIDE                                                           │
└────────────────────────────────────────────────────────────────────────────────┘

  MODE DÉMO (Sans base de données)
  ─────────────────────────────────
  
  1. cd LeadAssuranceSaas
  2. npm install
  3. npm run dev
  4. Ouvrir http://localhost:3000
  
  → Toutes les pages sont accessibles avec données mock
  → Parfait pour démonstration client


  MODE COMPLET (Avec base de données)
  ────────────────────────────────────
  
  1. Créer une base PostgreSQL (Supabase gratuit recommandé)
  2. Modifier DATABASE_URL dans .env
  3. npx prisma migrate dev
  4. npx prisma db seed
  5. npm run dev
  
  → Plateforme fonctionnelle avec vraies données
  → Comptes de test créés automatiquement


  SCRIPT DE DÉMARRAGE AUTOMATIQUE
  ────────────────────────────────
  
  Exécuter : ./quick-start.sh
